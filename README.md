# AI-Based Resume Analyzer

## 📌 Description
AI-Based Resume Analyzer is a Python application that extracts and analyzes resume content from PDF files to identify key skills, evaluate relevance, and generate a score. It provides actionable feedback to improve resumes.

---

## 🚀 Features
- Extracts text from PDF resumes  
- Identifies key skills  
- Generates resume score  
- Provides improvement suggestions  

---

## 🛠️ Tech Stack
- Python  
- PyPDF2  

---

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt

3. Run the program:
   python app.py
